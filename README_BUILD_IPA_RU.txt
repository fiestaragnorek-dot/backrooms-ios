Backrooms iOS — v5 streaming + locations + latestlog

Что добавлено:
1) latestlog.txt как в Minecraft:
   - пишется в Documents/latestlog.txt;
   - логируются AppDelegate, viewDidLoad, загрузка ресурсов, создание сцены, стриминг комнат, первый игровой тик;
   - включён File Sharing в Info.plist, чтобы latestlog.txt можно было забрать через Files/iTunes/Finder;
   - в главном меню есть кнопка LATESTLOG для просмотра лога прямо в игре.

2) Оптимизация мира через streaming 3x3:
   - одновременно существуют только 3x3 комнаты вокруг игрока;
   - комнаты сзади удаляются;
   - новые комнаты впереди создаются процедурно;
   - механики не вырезаны.

3) Новые типы комнат/проходов:
   - большие пустые комнаты;
   - комнаты-свалки с кучей мебели/металла по центру;
   - горка/скат вниз: игрок начинает быстро скользить;
   - комната без потолка с фейковым лабиринтом выше как декорация;
   - прямые лестницы вверх;
   - закруглённые/спиральные лестницы.

4) Сохранено:
   - двери;
   - ящики;
   - столы;
   - лампы;
   - трубы;
   - текстуры;
   - звуки;
   - UI.

Если всё равно вылетает сразу:
1. Установи/запусти приложение.
2. Открой Files/Файлы -> On My iPhone/На iPhone -> Backrooms.
3. Найди latestlog.txt.
4. Пришли содержимое latestlog.txt.

Сборка на Mac:
  brew install xcodegen
  chmod +x build_local_ipa.sh
  ./build_local_ipa.sh

Сборка через GitHub Actions:
  Залить проект -> Actions -> Build IPA -> скачать artifact Backrooms.
